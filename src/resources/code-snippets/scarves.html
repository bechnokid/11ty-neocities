--- title: Tumblr Scarves --- {% renderTemplate 'md' %} In 2012, there was a fad created by [Vanja](https://web.archive.org/web/20121002221021/http://asongofwaterearthfireandair.tumblr.com/scarves) where people decorated their blog with scarves to give others a glimpse into what media they liked and what fandoms they were in. I made a few of these myself! A few issues with the original code was that they usually only looked good on center-aligned blogs and wasn't entirely beginner-friendly since it required quite a bit of configuration on the user's part. However, I don't fault Vanja for this at all since a lot of modern CSS properties such as flex did not exist during that time. I think he did great with what was available at the time. So, after feeing nostalgic, I wanted to challenge myself by modernizing the code and also make the scarves entirely in CSS, allowing them to be fully customizable and fit almost any image (recommended size: 25px x 49px). {% endrenderTemplate %}
