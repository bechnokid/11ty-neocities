CSS

{% details { title: "(Click to show/hide code)", summaryCls: "details h4" } %} {% renderTemplate "md" %} ```css/ @keyframes marquee { 0% { transform: translateX(100%); } 100% { transform: translateX(-150%); } } .marquee-container { overflow: hidden; white-space: nowrap; width: 100%; } .marquee-content { display: inline-flex; gap: .5rem; animation: marquee 30s steps(400, end) infinite; } .marquee-content:hover { animation-play-state: paused; } .marquee-content img { vertical-align: middle; } ``` - **Lines 2-3**: `translateX` values ensure that all items in the example "disappear" before cycling through again. These numbers can change depending on how wide the marquee will be and how many items will be in it. Feel free to experiment! - For the marquee to scroll from left to right (as opposed to right to left in the example), reverse the numbers. (change 100% to -150%, and change -150% to 100%) - **Line 9**: this width can be any number. I just chose 100% so that it would span the width of this page. - **Line 14**: `gap` can only be used by flex boxes, and it gives some space between flex elements. This is optional and can be removed. - **Line 15**: `30s` can be changed to have the following effects: - Lower number = faster scrolling - Higher number = slower scrolling - **Line 15**: 400 in `steps()` can be changed to have the following effects: - Lower number = choppier movement - Higher number = smoother movement - **Lines 18-20**: Causes the marquee to pause at the user's cursor position when hovered over. This is optional and can be removed. {% endrenderTemplate %} {% enddetails %}
